Clones Help

In the clones section, you can manage all your antibody clones, and see clones from other groups you belong to. The information is structured in the following way Clone -> Lot -> Conjugate. Clones are the antibodies in your database, lots are the specific tubes for each antibody that you have purchased, purified, or obtained by any other means, conjugates are the aliquots of that clones that are labeled with metal isotopes or fluorophores. In this page, you can manage all this entities. The system will assign numbers to all lots and conjugates and you can use this numbers to keep your antibody stocks tidy and centralized for all the lab!! Lots are also considered reagents, as they can be purchased. Therefore, when a lot is added, it will appear as well in the "Inventory" section.

Clone options:

  • The button offers the following options
  • Add Validation

    Opens a dialog box where you can add information on the clone performance. Fill the form with all the data related to the experiment, including positive and negative control, and a file (PDF, image, etc) containing the results. This information will be attached to the clone and will be available to all the users in the group, and upon making a clone public, to the whole community.
  • Add Lot

    Adds a lot as explained above
  • Edit Clone

    Opens a pane to change the clone information
  • Delete Clone

    Deletes a clone. IMPORTANT!! Deleting a clone is irreversible, and all validation data associated will be lost, except the files uploaded, which will be still available in the system.
  • Make Public

    Makes the clone available to the AirLab community. The clone can now be searched in the AirLab portal.
  • Make Private

    Turns a clone private again, and only the group members can access it.

Lot options:

  • The button offers the following options
  • Edit Lot

    Opens a dialog box where you can edit the lot information
  • Add Conjugate

    Adds a conjugate as explained above
  • Reorder

    Opens a pane to change the clone information
  • Mark as low / Undo Mark as low

    Labels this tube as "Low" or reverses it to normal. This will warn everybody about this fact. This tube will show in red color everywhere in the system, including antibody panels.
  • Mark as finished / Undo Mark as finished

    Labels this tube as "Finished" or reverses it to non-finished. This will warn everybody about this fact. This tube will appeared in light-gray and crossed font everywhere in the database, including the antibody panels where it has been added.
