Panels Help

In the panels section, you can manage all your metal-labeled antibody panels for Mass Cytometry. This tool is simple to use yet really powerful. Create panels and add conjugates from the database isotope channel by isotope channel. You can then copy panels, quickly calculate volumes for pippeting them, export .conf file for CyTOF1, CyTOF2, or Helios, and much more. You can also see other members' of the group panels if the administrator has activated this option for you. You can not modify other people's panels, but you can create your own copy on a single click
